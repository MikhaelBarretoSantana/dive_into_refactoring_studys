# Dive Into Refactoring

Repositório de estudos do curso **Dive Into Refactoring**, feito a partir do conteúdo do **Refactoring Guru** e concluído como parte da minha prática guiada de refatoração de código.

Este projeto reúne exemplos práticos de antes e depois, organizados por técnica, para registrar a evolução do meu entendimento sobre design, legibilidade e redução de acoplamento. Ao longo do curso, aprendi a identificar cheiros de código com mais precisão, escolher refatorações menores e mais seguras, e melhorar a estrutura do código sem alterar o comportamento esperado.

## O que aprendi

- Como reconhecer problemas recorrentes de design e código duplicado.
- Como dividir responsabilidades para deixar classes e métodos mais coesos.
- Como substituir condicionais complexas por soluções mais claras.
- Como aplicar refatorações com passos pequenos para reduzir risco.
- Como usar herança, composição e delegação de forma mais equilibrada.
- Como tornar o código mais expressivo, previsível e fácil de manter.
